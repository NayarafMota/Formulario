# 🚀 React Hooks Context App

Aplicação desenvolvida em React utilizando Hooks, Context API e Styled Components.

## 📚 Funcionalidades

- Navegação entre páginas (Home, Profile, Contact e About)
- Gerenciamento de estado global com Context API
- Alteração do nome do usuário (Home)
- Alteração da idade do usuário (About)
- Contador interativo
- Componente Footer reutilizável
- Estilização com Styled Components

## 🧠 Tecnologias utilizadas

- React
- React Router DOM
- Styled Components
- Context API

## 📂 Estrutura

- Home
- Profile
- Contact
- About
- Components (Header, Footer, Title, Counter)
- Contexts (UserContext, AgeContext)
